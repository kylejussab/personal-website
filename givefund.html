BACKGROUND

Why?

Give Fund was our final project at the end of the 9-week Snap Academies program in Summer 2024, where we were challenged to design a feature that could make a meaningful impact through Snap’s platform. In the final 3 weeks, our team set out to prototype a real solution tied to a cause we cared about.

We chose to focus on foster youth nonprofits, organizations that are often underfunded and underrepresented. We spoke directly with nonprofit workers and researched what Snapchat could uniquely offer. What we heard consistently: nonprofits needed both visibility and accessibility when it came to donations.

That led us to ask: What if giving didn’t have to cost anything? What if Snap users could support causes by watching sponsored content and converting ad revenue into donations? Give Fund was born out of that idea, a way to blend community, design, and generosity into a single feature, built for the people who needed it most.

    const [amount, setAmount] = useState(0);
    const [crowdImage, setCrowdImage] = useState(require("../../../assets/crowdWithout.png"));
    
    ///SUPABASE CALL
    async function fetchNonprofits() {
    try {
        const { data, error } = await supabase
        .from("nonprofits")
        .select("*")
        .eq("registrationNumber", id);
        if (error) {
        console.error("Error fetching nonprofits:", error.message);
        return;
        }
        if (data) {
        setNonprofits(data);
        setFollowNum(data[0].followers);
        setAmount(data[0].currentAmount);
        if(data[0].hasDonated){
            setCrowdImage(require("../../../assets/crowdWithMariah.png"))
        }
        }
    } catch (error) {
        console.error("Error fetching Nonprofits:", error.message);
    }
    }

    useEffect(() => {
    fetchNonprofits();
    }, [originScreen]);

    setBadgeSelected(!badgeSelected);

    async function fetchNonprofits() {
